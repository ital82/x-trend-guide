改版履歴

2026-05-10

  1. 検索キーワードの拡充: 6件→11件に増やし、収集量を約6倍に向上
  2. ランキング精度の改善: いいね数が多い投稿が下位になる問題を修正
  3. 宣伝投稿の除外強化: 「♡でお迎え」「DMで見せ」などの自己宣伝・誘導投稿を自動で除外
  4. いいねが少なくても良い投稿を残す: あかりさんに合う投稿は、いいねが少なくても上位に表示
  5. 流行フォーマット検出の精度向上: あかりさんの活動と関係ない流行を除外

2026-05-13

  1. 大バズ投稿を上位に: いいね数万件の投稿がきちんと上位に来るよう調整
  2. 速報の中身を絞り込み: 「あかりさんが乗れるかどうか」を基準に絞り込むように変更
  3. 速報の頻度を整理: 1日4回→2回（朝・夕のみ）に集約

2026-05-19

  1. 同じ人の投稿ばかり並ぶ問題の解消: 同じ人の投稿が2件目以降は順位が下がるよう調整
  2. 写真提案の改善: 顔出しなし前提で、体のアングル・衣装・構図で提案するように変更
  3. Xの表示ルール分析に基づく改善: 公開されたXの表示ルールを分析し、投稿戦略の精度を向上

重要2026-05-29（大型アップデート）

[変更点]引用RT候補の品質改善

  1. 海外の流行投稿の混入を排除: 英語・中国語の投稿が上位に居座る問題を解消。日本語投稿のみが並ぶように
  2. 「OL」「JD」の誤判定を修正: 英単語「Olivia」「goldship」などに含まれる "OL" を OL（女性社員）と勘違いしていた問題を解消
  3. ゲーム公式アカウントの除外: 原神公式などのキャラクター告知投稿を、アカウントごと候補から除外

[変更点]投稿ネタ提案の改善

  1. 胸ネタばかり提案される問題の解消: AIへの指示文に固定で書き込まれていた「胸の話題は確実にバズる」等の表現を削除。色々なジャンル（コスプレ・裏垢女子・ナース等）からも提案されるように
  2. 表示制限関連の表記削除: 不安につながりやすい記述を削除
  3. 提案ネタの差別化: 「3つの提案ネタは別々のジャンル・写真構図にする」ことを指示文に追加し、同じテーマでの連投を回避

[変更点]調査対象アカウント一覧の作り直し

  1. ジャンル別 ○/× 判定形式に変更: 全件を機械的に並べる形式から、ジャンル別10件ずつ × 7ジャンルへ
    （巨乳・体型系 / コスプレ系 / 裏垢女子 / ナース / 保育士 / 大学生・JD / OL・社会人 / その他）
    ※当初は「グラビア・モデル」ジャンルも候補に含めていましたが、裏垢女子クラスタとの親和性を検証した結果、対象アカウントの大半が出版社・雑誌の公式アカウントや事務所所属タレントとなり、あかりさんの引用RT戦略に合わない内容が中心と判明したため、追加を見送りました。
  2. ○で採用／×で対象外: 判定欄に記号を入れるだけで、次回からシステムに反映
  3. ランダムに表示: 同じ日のうちは同じ顔ぶれ、日替わりで違う顔ぶれが見られる仕組み
  4. ★必須マーク: あかりさんが「監視アカウント追加」シートで指定したアカウントは★必須として各ジャンルの先頭に常時表示。誤って×を付けても消えない保護機能つき

[変更点]監視アカウント自動選定の見直し

  1. 公式アカウントの自動除外: プロフィール文に「【公式】」「ニュース」「Patriot」「株式会社」等が含まれるアカウントは、自動で監視対象にしないように
  2. 無関係アカウントの排除: 「配信」というキーワードが、ニュースサイトの「配信中」「ニュース配信」に誤判定されていた問題を解消
  3. 海外コスプレ系の救済: 海外コスプレイヤーの英語表記（"cosplay" "cosplayer"）も認識されるように

[変更点]監視アカウントの自動発見

  1. 「監視アカウント追加」シートを発見の起点に: あかりさんが追加したアカウントの「Xのおすすめ」欄から、似たアカウントを自動で見つける仕組みに変更。シートに追加するだけで、自動的に対象が広がる

[変更点]ジャンル別アカウント収集の強化

  1. ナース・看護師・保育士の裏垢収集を強化: 「ナース 裏垢」「保育士 裏アカ」など、複数キーワードを組み合わせた検索を7パターン追加

[変更点]その他

  1. 監視対象に入れる条件を厳しく: フォロワー数が多いだけで自動採用される問題を解消（プロフィール文にテーマキーワードが含まれることが必須に）
  2. プロフィール文の再取得: 必須アカウントでプロフィール文が取れていないものは、次回の自動収集で取り直し
